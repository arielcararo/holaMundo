# Hola Mundo App 🌍

Una aplicación web moderna y interactiva que demuestra los conceptos básicos de desarrollo web con HTML, CSS y JavaScript.

## ✨ Características

- **Diseño moderno**: Interfaz atractiva con gradientes y animaciones
- **Interactividad**: Botones funcionales con efectos visuales
- **Responsive**: Se adapta a diferentes tamaños de pantalla
- **Contador de clicks**: Registra las interacciones del usuario
- **Múltiples mensajes**: 10 mensajes diferentes que se pueden alternar

## 🚀 Cómo usar

1. Abre el archivo `index.html` en tu navegador web
2. Haz clic en "Cambiar Mensaje" para ver diferentes mensajes
3. Usa "Reiniciar" para volver al estado inicial
4. Observa el contador de clicks que se actualiza
