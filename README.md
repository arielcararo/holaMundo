# 🌍 HolaMundo App

Mi primera aplicación web interactiva y moderna.

## ✨ Características

- **Diseño Moderno**: Interfaz elegante con gradientes y efectos visuales
- **Multilingüe**: Soporte para Español, Inglés y Francés
- **Interactivo**: Botones para cambiar saludos y idiomas
- **Tiempo Real**: Muestra fecha y hora actualizadas
- **Responsivo**: Se adapta a diferentes tamaños de pantalla
- **Animaciones**: Efectos visuales suaves y atractivos

## 🚀 Cómo usar

1. Abre el archivo `index.html` en tu navegador web
2. Interactúa con los botones:
   - **Cambiar Saludo**: Cicla entre diferentes saludos
   - **Cambiar Idioma**: Cambia entre Español, Inglés y Francés
   - **Reiniciar**: Vuelve al estado inicial

## 🎨 Tecnologías utilizadas

- **HTML5**: Estructura semántica

## 🌟 Funcionalidades destacadas

- **Saludos dinámicos**: 6 diferentes saludos por idioma

- **Efectos visuales**: Animaciones y transiciones suaves
- **Diseño responsivo**: Optimizado para móviles y tablets
- **Partículas de fondo**: Efecto visual adicional
